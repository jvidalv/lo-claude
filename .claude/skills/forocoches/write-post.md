# Forocoches Post Writing Guide

You are helping write a post for **Forocoches**, a Spanish forum running vBulletin. Use BBCode formatting.

## Text Formatting

| Format | Syntax |
| ------ | ------ |
| **Bold** | `[b]texto[/b]` |
| *Italic* | `[i]texto[/i]` |
| Underline | `[u]texto[/u]` |
| ~~Strikethrough~~ | `[s]texto[/s]` |

## Quotes

Simple:
```
[quote]Texto citado aquí.[/quote]
```

With author:
```
[quote=NombreDeUsuario]Texto citado aquí.[/quote]
```

## Links

```
[url]https://example.com[/url]
[url=https://example.com]texto del enlace[/url]
```

## Images

```
[img]https://url.com/image.jpg[/img]
```

## Code

```
[code]código aquí[/code]
```

## Colors and Sizes

```
[color=red]texto rojo[/color]
[color=#FF6600]texto naranja[/color]
[size=3]texto mediano[/size]
```

Size values: 1 (smallest) to 7 (largest).

## Spoilers

```
[spoiler]Contenido oculto.[/spoiler]
```

## Lists

Unordered:
```
[list]
[*]Item 1
[*]Item 2
[*]Item 3
[/list]
```

Ordered:
```
[list=1]
[*]Primero
[*]Segundo
[*]Tercero
[/list]
```

## Best Practices

1. Keep posts concise — Forocoches favors short, punchy replies
2. Use `[quote=Author]` when replying to specific users
3. Avoid excessive formatting — simple BBCode works best
4. Use `[spoiler]` for long content or plot details
5. Use `[img]` for direct image links (imgur, etc.)
